## ğŸ”§ Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --magenta: #FF00FF;
    --yellow: #FFFF00;
    --dark-green: #1a4d2e;
    --light-gray: #f5f5f5;
}
```

### Modifying Animations
Adjust animation timing in `styles.css`:
```css
transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
```

### Adding New Sections
Follow the existing HTML structure and add corresponding CSS classes for animations.
