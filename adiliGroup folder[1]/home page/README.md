# Adili Group Homepage

## Overview
This is a pixel-perfect implementation of the Adili Group homepage based on the provided design specifications.

## Files Structure
- `index.html` - Main HTML file
- `styles.css` - Complete CSS styling with exact dimensions
- `Images Used/` - All images for the website
- `Fonts Used/` - Cascadia Pro and Jost fonts
- `Icons Used/` - Social media icons

## Design Specifications

### Colors
- **Magenta**: #FF00FF (accent bars and dimension labels)
- **Yellow**: #FFFF00 (highlight bars and dimension labels)
- **Dark Green**: #1a4d2e (deadline section background)

### Typography
- **Cascadia Pro**: Used for dimension labels and technical text
- **Jost**: Main font family
  - Regular (400): Body text (22px)
  - Semi Light (300): Body text variations
  - Bold (700): Headings and action buttons

### Font Sizes
- 22px: Body text
- 24px: Action buttons (e.g., "Learn More")
- 36px: Section headings
- 72px: Hero and major headings

### Key Dimensions
- W96: Left magenta bar width, navigation spacing
- W144: Service card left spacing
- W422: Yellow bar width (narrow)
- W574: Yellow bar width (medium)
- W1920: Full width yellow bar
- H48: Navigation height, magenta bars
- H120: Spacing elements
- H240: Section magenta bars
- H512: Service card height
- H534: Footer right bar height
- H720: Hero sections height
- H962: Main hero section height

## Browser Compatibility
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design included for tablet and mobile

## How to View
Open `index.html` in a web browser to view the website.
